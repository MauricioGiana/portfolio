# Mauricio Giana | Backend Engineer Portfolio

This repository hosts my personal portfolio. It is engineered from scratch as a high-performance, lightweight, and responsive single-page resume tailored to showcase my technical stack, systems mindset, and architectural case studies.

## 🚀 Key Engineering Highlights

- **Semantic & Accessible Structure:** Built completely from scratch with strict semantic HTML5 tags and clean document hierarchy.
- **Zero-Dependency Core Interactions:** The accordion component for case studies is built using vanilla JavaScript, utilizing unique runtime optimizations (such as `requestAnimationFrame` double-cycling and `scrollHeight` layout calculations) to guarantee fluid max-height transition animations.
- **Optimized for Delivery:** Styled using Tailwind CSS via a utility-first methodology to minimize layout paint times and ensure precise visual scalability across all device screens.
- **Production-Ready Layout:** Designed for near-instant rendering speeds with zero bloated JS frameworks or unnecessary client-side overhead.

## 🛠️ Technical Stack

- **Markup:** Semantic HTML5 & DOM API
- **Styling:** Tailwind CSS (Responsive Grid & Flex layouts)
- **Scripting:** Pure ES6+ JavaScript (Vanilla)
